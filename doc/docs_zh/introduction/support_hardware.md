# 支持硬件

## ARM CPU

Tengine 已完成对下列 ARM Cortex-A 系列处理器支持：

### 32bit
- Cortex-A5
- Cortex-A7
- Cortex-A8
- Cortex-A9
- Cortex-A12
- Cortex-A15
- Cortex-A17
- Cortex-A32

### 64bit
- Cortex-A53
- Cortex-A55
- Cortex-A57
- Cortex-A72
- Cortex-A73
- Cortex-A75
- Cortex-A76

## 端侧 GPU

通过 ACL 已验证 ARM Mali-GPU 以下系列：

- T860
- G31
- G52

通过 CUDA/TensorRT 已验证 NVIDIA 以下系列：

- Jetson NANO
- Jetson TX1
- Jetson TX2
- Jetson XAVIER NX
- Jetson XAVIER AGX

## 服务器 GPU

通过 CUDA/TensorRT 已验证 NVIDIA 以下系列：

- QUADRO RTX 8000
- GeForce RTX 3090
- GeForce GTX 1080Ti

## NPU

通过 TIM-VX 已验证内置 VeriSilicon VIP8000/VIP9000 NPU 的以下芯片：

- A311D
- S905D3
- iMX.8MP
- JLQ
